Just setting things up, practicing pushing changes using Git

Git commands

  • 1.Create repo in Github
  • 2.Clone repo with 
    git clone http://github.com/url/to/repo
  • 3.
    cd
    to cloned repo folder
  • 4.Make stuff
  • 5.
    git add .
    to add all files and 
    git commit -m "Comment"
    to commit changes to repo
  • 6.And 
    git push http://github.com/url/to/repo -branch_name
