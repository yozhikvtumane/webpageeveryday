Git commands

  • 1.Create repo in Github
  • 2.Clone repo with 
    git clone http://github.com/url/to/repo
  • 3.
    cd
    to cloned repo folder
  • 4.Make stuff
  • 5.
    git add .
    to add all files and 
    git commit -m "Comment"
    to commit changes to repo
  • 6.And 
    git push http://github.com/url/to/repo -branch_name

Here is the solution about how I made those random cats pictures

So, it's very easy. All you have to do is to paste this url 

"http://lorempixel.com/300/300/cats"
into src of your image eg 
img src="http://lorempixel.com/300/300/cats" alt="Cats"

That numbers in link are 
width
and 
height
of your choise. And 
category
in the end, in this case is 
cats
